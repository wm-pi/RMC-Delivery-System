# BAD_EXAMPLES.md

이 문서는 프로젝트에서 피해야 할 실패 사례와 금지 패턴을 진행 중에 모으는 문서다.

프로젝트 시작 시 억지로 예시를 채우지 않는다. 실제 버그, 회귀, 보안 위험, 반복 실수가 확인되면 추가한다.

---

## 작성 시점

- 같은 실수가 반복될 때
- 보안, 권한, 데이터 손실, 배포 사고 위험이 발견될 때
- AI가 자주 잘못 생성하는 코드 패턴이 확인될 때
- 기존 구현 중 더 이상 따라 하면 안 되는 패턴이 생길 때

---

Anti-pattern:
